# E2E Smoke Scenarios

1. Launch app and verify top bar health appears.
2. Connect to configured OpenClaw instance and receive `instance.ready`.
3. Create task in Inbox and confirm persistence after restart.
4. Delegate task via CSO and confirm assignment to specialist.
5. Move task through kanban statuses to Done.
6. Send chat to CSO and receive streamed response.
7. Send direct `@agent` chat and confirm routing.
8. Trigger permission request and approve it.
9. View logs with filter and export log file.
10. Verify daily cost rollup updates after usage event.
