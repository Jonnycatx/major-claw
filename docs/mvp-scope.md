# MVP Scope Guardrails

## Must Have

- Connect to existing local OpenClaw instances.
- CSO with deterministic delegation and task state transitions.
- Hierarchical agent tree, kanban, unified chat and logs.
- Per-agent model assignment with fallback.
- Permission prompts and local audit logs.

## Deferred

- Cloud sync and remote control.
- Marketplace distribution.
- Voice features.
- Mobile companion.
- Visual workflow builder.
