# Major Claw Quickstart (5 Minutes)

## 1) Configure

- Copy `.majorclaw.instances.example.json` to `.majorclaw.instances.json`.
- Set your local OpenClaw WebSocket URL.
- Copy `.env.example` to `.env.local`.

## 2) Install and Run

- Install dependencies: `pnpm install`
- Run workspace: `pnpm dev:all`

## 3) Verify

- Open Mission Control UI.
- Confirm `CSO` and specialist agents appear in hierarchy.
- Confirm health is online and at least one instance is connected.

## 4) First Task

- Create a task in Inbox.
- Delegate through CSO.
- Move task to Review and Done.
