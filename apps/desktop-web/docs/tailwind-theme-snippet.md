# Tailwind Theme Snippet

Use this snippet in `tailwind.config.ts`:

```ts
extend: {
  colors: {
    void: "#0a0a0a",
    lobster: "#ff3b00",
    cyan: "#00f0ff",
    text: {
      primary: "#f1f1f1",
      secondary: "#a1a1aa"
    },
    panel: "#111113"
  },
  fontFamily: {
    sans: ["Inter", "-apple-system", "BlinkMacSystemFont", "sans-serif"],
    mono: ["SF Mono", "JetBrains Mono", "ui-monospace", "monospace"]
  }
}
```
